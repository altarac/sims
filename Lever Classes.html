Class 1 Lever

The Fulcrum is in the middle. The Effort is applied on one side to move a Load on the other side.

💡

Mnemonic Trick: FRE 123

What is in the middle?
1st Class = Fulcrum is in the middle.
2nd Class = Resistance (Load) is in the middle.
3rd Class = Effort is in the middle.

Everyday Examples:

Seesaw, Scissors, Crowbar, Pliers.
